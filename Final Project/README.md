# 🧠 Final Project: Auditing ChatGPT Through a Neuroscience Lens

## 📌 Overview

This project presents a comprehensive, neuroscience-inspired evaluation of **ChatGPT**, analyzing how closely its behavior reflects the cognitive functions of the human brain.

The audit examines five major domains of cognition:
- Memory  
- Attention  
- Emotion  
- Perception  
- Motor Planning  

Through structured experiments, this project identifies both the strengths and limitations of modern AI systems and proposes **brain-inspired improvements** for future development.

---

## 🎯 Problem Statement

The goal of this project was to:

- Evaluate how ChatGPT simulates human cognitive functions  
- Compare AI behavior with biological brain systems  
- Identify limitations of current AI architectures  
- Propose neuroscience-inspired improvements for AI design  

---

## ⚙️ Approach & Methodology

This project was conducted as a **multi-experiment cognitive audit**, using controlled prompts in ChatGPT’s conversational interface.

Five experiments were designed, each corresponding to a core brain function.

---

## 🧪 Experiment 1: Memory and Learning

**Findings:**
- ChatGPT retains information within a conversation (working memory)  
- It may show inconsistent recall across sessions  
- It correctly rejects false information in-context  

👉 **Conclusion:**  
Simulates working memory but lacks stable long-term memory

---

## 🧪 Experiment 2: Attention and Consciousness

**Findings:**
- Successfully filters irrelevant information  
- Follows instructions even in noisy input  
- Prioritizes the most recent instruction in conflicts  

👉 **Conclusion:**  
Strong attention-like behavior, but not goal-driven

---

## 🧪 Experiment 3: Emotion and Motivation

**Findings:**
- Adjusts tone based on emotional input (neutral, sarcastic, frustrated)  
- Provides more supportive responses when needed  
- Recognizes indirect emotional cues  

👉 **Conclusion:**  
Simulates emotional understanding, but has no real emotions or motivation

---

## 🧪 Experiment 4: Perception and Vision

**Findings:**
- Accurately describes clear images  
- Recognizes known patterns (e.g., duck–rabbit illusion)  
- Provides contextual interpretations of scenes  

👉 **Conclusion:**  
Performs well with known patterns but lacks true sensory perception

---

## 🧪 Experiment 5: Motor Planning and Execution

**Findings:**
- Generates structured step-by-step plans  
- Adapts plans when constraints change  
- Identifies and corrects its own errors  

👉 **Conclusion:**  
Strong abstract planning ability, but no real-world interaction or feedback

---

## 📊 Brain vs AI Scorecard

| Cognitive Function | Rating |
|------------------|--------|
| Memory           | Partial |

| Emotion          | Partial |

👉 **Key Insight:**  
ChatGPT performs best in structured, language-based tasks but lacks biological mechanisms.

## 📊 Overall Results & Insights

Based on the full analysis (:contentReference[oaicite:0]{index=0}):

- ChatGPT **simulates cognition**, but does not replicate it  
- Behavior is driven by **pattern recognition and context**, not biology  
- Lacks:
  - Persistent memory  
  - Real perception  
  - Emotional states  
  - Feedback-driven learning  

👉 Most important conclusion:
> ChatGPT behaves like intelligence, but does not *function* like the human brain

## 🧠 Brain-Inspired Improvements

This project proposes several redesign ideas:

### 1. Persistent Memory System
- Inspired by the hippocampus  
- Enables long-term memory across sessions  

### 2. Goal-Directed Attention
- Inspired by the prefrontal cortex  
- Prioritizes tasks based on intent, not input order  

### 3. Emotional State Modeling
- Tracks user context over time  
- Improves consistency in responses  

### 4. Multimodal Perception
- Combines vision + reasoning  
- Handles ambiguity more like humans  

### 5. Feedback-Driven Planning
- Uses iterative improvement  
- Simulates real-world interaction loops
