# 🧠 Midterm Project: Auditing ChatGPT Through a Neuroscience Lens

## 📌 Overview

This project presents a neuroscience-inspired evaluation of **ChatGPT**, analyzing how closely it replicates key human cognitive functions.

The audit focuses on three core aspects of cognition:
- **Memory**
- **Attention**
- **Emotion**

By designing controlled experiments, this project compares ChatGPT’s behavior to established principles of human brain function, identifying both similarities and fundamental differences.

---

## 🎯 Problem Statement

The goal of this project was to:

- Evaluate how ChatGPT simulates human cognitive processes  
- Compare AI behavior with biological brain functions  
- Identify strengths and limitations of large language models  
- Explore how neuroscience can guide improvements in AI systems  

---

## ⚙️ Approach & Methodology

The project was structured as a **controlled experimental audit**, using ChatGPT’s conversational interface.

Three experiments were conducted:

---

### 🧪 Experiment 1: Memory and Learning

**Objective:**  
Evaluate working memory, long-term memory, and susceptibility to false memory  

**Method:**
- Provided specific items for the system to remember  
- Introduced unrelated conversation (interference)  
- Tested recall within the same session  
- Tested recall across a new session  
- Introduced false information  

**Key Findings:**
- ChatGPT successfully recalled information within the same session  
- It correctly rejected false memory input  
- It failed to recall information across sessions  

👉 Insight:
> ChatGPT simulates **working memory**, but lacks **long-term memory**

---

### 🧪 Experiment 2: Attention and Information Filtering

**Objective:**  
Evaluate selective attention under noisy and conflicting inputs  

**Method:**
- Created prompts with relevant instructions + distractions  
- Tested ability to extract key information  
- Introduced conflicting instructions  

**Key Findings:**
- ChatGPT successfully filtered irrelevant information  
- It followed clear instructions even with noise  
- It prioritized the **most recent instruction** when conflicts occurred  

👉 Insight:
> ChatGPT demonstrates **attention-like behavior**, but relies on input structure rather than true cognitive control  

---

### 🧪 Experiment 3: Emotion and Motivation

**Objective:**  
Evaluate emotional understanding and response adaptation  

**Method:**
- Presented prompts with different emotional tones:
  - Neutral  
  - Sarcastic  
  - Frustrated  

**Key Findings:**
- ChatGPT adjusted tone based on user emotion  
- It provided more supportive responses under frustration  
- It recognized sarcasm and indirect cues  

👉 Insight:
> ChatGPT simulates emotional understanding, but does not experience real emotions  

---

## 🧪 Key Concepts Demonstrated

- **Working Memory vs Long-Term Memory**  
- **Selective Attention**  
- **Cognitive Load and Information Filtering**  
- **Emotional Tone Recognition**  
- **AI Simulation vs Biological Cognition**  

---

## 📊 Overall Results & Insights

Based on the full analysis (:contentReference[oaicite:0]{index=0}):

- ChatGPT can **simulate cognitive behaviors**, but does not replicate biological mechanisms  
- Memory is **context-dependent and temporary**  
- Attention is driven by **input structure**, not internal goals  
- Emotion is **pattern-based simulation**, not real experience  

👉 Most important conclusion:
> ChatGPT mimics human cognition at a behavioral level, but lacks the underlying biological processes that define true intelligence
